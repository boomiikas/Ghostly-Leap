**Ghostly Leap** is a one-tap atmospheric WebGL game built using **p5.js**, where players guide a lost spirit across haunted bridges by matching soul-taps. Created for casual and spooky arcade gameplay, it features immersive visuals, parallax backdrops, and smooth progression across levels.

## ğŸŒ‰ Game Description

In *Ghostly Leap*, you play as a friendly ghost drifting through a mysterious haunted world. Each bridge ahead is locked with a magical count of souls. Tap the screen the **exact number of times** shown by the floating orbs to leap ahead. Miss the count and your ghostly journey ends.

The game features dynamic backdrops with glowing moons, drifting mist, bats, gravestones, and soul-fireflies, delivering an eerie but calming experience.
